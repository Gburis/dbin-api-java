# Projeto - DBIN API (Cidades ESG Inteligentes)
 API desenvolvida em Java com Spring Boot para gestão de resíduos e reciclagem, seguindo práticas de DevOps com containerização e CI/CD.

## Pré-requisitos

- Java 21
- Git
- Docker

## Build e execução

```sh
docker-compose up --build
```

## Testes unitários (validação)

./mvnw test
